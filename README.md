# Reference Interaction Site Model for molecular fluids #


## The C program `rism0` ##

`rism0` is an XRISM program.
See the [prog/README.md](prog/README.md) under program for details.

### Features ###

#### Solvers ####
 * Picard solver
 * Labik-Malijevsky-Vonka solver
 * modified (MDIIS) solver

#### Closures ####
 * PY
 * HNC
 * KH
